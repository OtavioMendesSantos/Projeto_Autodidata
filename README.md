# Regras de Formatação de nomes de Arquivos 
 - Sem Caracteres Especiais
 - Alfabeto inglês (a a z e A a Z)
 - Números (0 a 9)
 - Ponto final (.)
 - Hífen (-)
 - Sublinhado (_)
 - Barra (/)
