Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b>, ou seja, uma tag não semântica.

Nesta frase, temos um termo em destaque usando a tag <strong>, ou seja, uma tag semântica.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>, ou seja, uma tag não semântica

Nesta frase, temos um termo em ênfase usando a tag <EM>, ou seja, uma tag semântica

Texto Marcado

Podemos também, criar um texto marcado usando a tag <mark>

Podemos támbem, mudar a cor do Texto Marcado utilizando Css.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo, utilizando a tag <big> (obsoleta) e <small>, ambas tags não semânticas

Texto Deletado

Podemos marcar que um texto foi deletado, que pode ser lido, e não deve ser considerado.

Texto inserido

Podemos marcar um texto como inserido para se dar ênfase e indicar que foi adicionado depois.

Existe também a Tag <U>(não semântica)
